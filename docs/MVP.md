# job-search-automation

## 1. Problem
I want a system that collects frontend / UI / UX / frontend jobs in Sweden and shows them in one place.

## 2. MVP Goal
Build a working system that:

- fetches job listings
- stores them in a database
- displays them in a simple dashboard

## 3. In scope (VERY important)
In scope:
- Arbetsförmedlingen job API
- SQLite database
- FastAPI backend
- Next.js frontend
- simple job list UI

## 4. Out of scope (equally important)
Out of scope (for now):
- AI scoring
- CV generation
- cover letters
- LinkedIn automation
- Playwright automation
- login system
